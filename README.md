🧮 Simple Calculator in C++

📌 Descrição

Este projeto é uma calculadora simples desenvolvida em C++ que permite ao usuário realizar operações matemáticas básicas diretamente pelo terminal.

🖥️ O programa suporta:
- Adição (+)
- Subtração (-)
- Multiplicação (*)
- Divisão (/)


🚀 Como funciona
- O usuário insere uma operação no formato numero -> operador -> numero
- Exemplo: 10 + 5
- O programa então calcula e exibe o resultado. Após isso, é possível escolher se deseja realizar outra operação.


🛠️ Tecnologias utilizadas
- C++
- Biblioteca padrão <iostream>


▶️ Como executar
- Compile o código: *g++ calculadora.cpp -o calculadora*
- Execute: *./calculadora*


📋 Exemplo de uso
- Escreva uma conta: 8 * 3
- Resultado: 24
- Fazer outra operação? (s/n): s
- Escreva uma conta: 10 / 0
- Erro: Divisão por zero!
- Fazer outra operação? (s/n): n


⚠️ Tratamento de erros
- Não realiza divisão por zero
- Não aceita operadores inválidos
